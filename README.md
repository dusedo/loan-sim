# ローン返済 ＆ 資産運用シミュレーター

自分用の住宅ローン返済・資産運用シミュレーター（単一HTML・外部依存なし）。

- **ローン返済**：全期間固定・元利均等。借入額/金利/期間をスライダー＋バッジで調整。フラット35S・子育てプラスのポイント制金利引き下げ（多段ステップ）、繰上返済、複数プラン比較、償却表、完済年月表示。
- **資産運用**：NISA（非課税）／特定口座（20.315%課税）／iDeCo（手数料・所得控除考慮）で手取り受取額を比較。毎月積立・想定年利・信託報酬を考慮した月複利。
- **比較**：同じ毎月額を「繰上返済の利息軽減」と「投資の手取り運用益」で見比べ。

スライダー操作でグラフがリアルタイムに動き、ホバーで数値を表示。入力は端末ごとに localStorage に自動保存。

## 使い方
`index.html` をブラウザで開くだけ（GitHub Pages で公開時はそのURL）。

> 試算は概算です。実際の返済額・利息・運用結果は金融機関や市場により異なります。
